## Working directory
All commands run from the project root using relative paths.

## Safety
The external HTTP sources (Polymarket, Odds API, Kalshi, ESPN, FBref, eloratings) are read-only
and several need no key. To avoid live-network flakiness and free-tier budget, the **default
integration run is offline**: drive the orchestrator against a captured snapshot, then verify the
exported artifacts. A live smoke run is optional and explicitly gated.

## Pre-flight
1. `python -c "import numpy, requests"` — deps present.
2. `python -m pytest quality/test_functional.py tests/test_model.py -q` — unit safety net green.

## Test matrix
| Group | UC | What it exercises | Command | Pass criteria |
|-------|----|-------------------|---------|---------------|
| 1 | UC-01 | Full pipeline produces a snapshot + dashboard JSON | `python -m src.pipeline.daily_matchday_run --skip-whales` | exits 0; `data/snapshots/<ts>.json` and `data/processed/dashboard_data.json` written; ≥0 matches, no traceback in `run_log.matches_failed` for network-available matches |
| 2 | UC-01 | Static export self-contained | inspect `docs/index.html` after group 1 | file present; embedded JSON parses; no `</script>` breakout |
| 3 | UC-02 | Value/opportunity portfolio shape | read `value_portfolio`/`opportunity_portfolio` in the snapshot | `total_stake_pct ≤ PORTFOLIO_CAP`; per-match ≤ `PER_MATCH_CAP` |
| 4 | UC-05 | Calibration resolves forecasts | read `calibration` in snapshot | `status` reported; if `live`, summary has Brier/RPS per source; no leakage (all forecasts pre-kickoff) |
| 5 | [Infrastructure] | Unit suite regression guard | `python -m pytest -q` | all pass |

## Execution UX
(1) print the matrix as a numbered table; (2) one-line progress per group (✓/✗/⧗);
(3) summary table + recommendation (SHIP / FIX BEFORE MERGE / BLOCK).

## Notes
Because most matches depend on live markets, group 1 may legitimately find 0 open events
out of season; that is a SKIP, not a failure. The offline-snapshot path is the deterministic
core; live runs are advisory.
