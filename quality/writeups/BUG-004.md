# BUG-004 — Ganzzahlige Totals-Linie schlaegt Push der Under-Seite zu

> Generated by Quality Playbook v1.5.6 — https://github.com/andrewstellman/quality-playbook
> Date: 2026-06-14 · Project: wm-dashboard-fussball-wahrscheinlichkeit · Severity: MEDIUM · REQ-004

## 1. Summary
Ganzzahlige Totals-Linie schlaegt Push der Under-Seite zu.

## 2. Spec reference
REQ-004 — p_over + p_under + p_push = 1.

## 3. The code
p_under = 1 - p_over mit p_over = 1 - sum(total_dist[:int(ln)+1]). Bei ganzzahliger Linie wird P(total==ln) ganz dem Under zugeschlagen. (value_betting.py:143-145)

## 4. Observable consequence
Linie 3.0: Push P(total=3)=0.214; Code p_under=0.734 statt korrekt P(total<=2)=0.520. Under-EV nach oben verzerrt. .5-Linien unbetroffen.

## 6. The fix
```diff
--- a/src/model/value_betting.py
+++ b/src/model/value_betting.py
@@ (import math noetig)
-        floor = int(ln)
-        p_over = round(1.0 - sum(total_dist[: floor + 1]), 4)
-        p_under = round(1.0 - p_over, 4)
+        p_over = round(1.0 - sum(total_dist[: int(ln) + 1]), 4)       # P(total > ln)
+        p_under = round(sum(total_dist[: int(math.ceil(ln))]), 4)     # P(total < ln)
```
