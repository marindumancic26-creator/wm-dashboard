## 1. Summary
Data-Quality market_agreement ignoriert draw/away.

## 2. Spec reference
REQ-008 — Agreement muss alle drei Ausgaenge beruecksichtigen.

## 3. The code
Dispersion nur ueber probs['team1_win'] (data_quality.py:24-35).

## 4. Observable consequence
Quellen einig bei team1_win=0.50 aber draw/away vertauscht -> Agreement 1.000. Tier ueberschaetzt -> Einsatz-Gate zu optimistisch. LOW.

## 6. The fix
```diff
--- a/src/model/data_quality.py
+++ b/src/model/data_quality.py
@@ def _market_agreement(match):
-    probs = [match[s]['probs']['team1_win'] for s in ('market','books','kalshi') if match.get(s) and match[s].get('probs')]
-    disp = statistics.pstdev(probs)
+    rows = [match[s]['probs'] for s in ('market','books','kalshi') if match.get(s) and match[s].get('probs')]
+    disp = max(statistics.pstdev([r[o] for r in rows]) for o in ('team1_win','draw','team2_win'))
```
