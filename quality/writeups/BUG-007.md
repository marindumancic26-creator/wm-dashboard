## 1. Summary
Ensemble whale-Default 0.2 != config 0.15.

## 2. Spec reference
REQ-009 — Inline-Defaults muessen config.ENSEMBLE_WEIGHTS entsprechen.

## 3. The code

## 4. Observable consequence

## 6. The fix

--- a/src/model/ensemble.py
+++ b/src/model/ensemble.py
